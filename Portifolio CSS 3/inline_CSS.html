Inline

inline é usado para dar estilo a um elemento HTML específico. Para este estilo de CSS < você somente vai precisar adicionar o atributo style para cada tag HTML, sem usar os seletores. Este tipo de CSS não é realmente recomendado, já que cada tag HTML precisa ser estilizada de maneira individual.
