Div

O elemento div é semanticamente neutro: ele não tem qualquer valor semântico, é simplesmente um elemento container. O elemento div deverá ser usado quando não estiver disponível um outro elemento container e semântico, capaz de substituí-lo tal como são os cabeçalhos, parágrafos, listas etc.

Exemplo

  • nome: lorenzo
  • idade: 20 anos
  • profissão: desenvolvedor web
  • nacionalidade: brasileiro
  • estado:rio grande do sul
  • cidade natal: porto alegre
